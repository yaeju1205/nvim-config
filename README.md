windows 와 linux 를 동시에 사용하기 때문에 어떤 os 에 맞춘 설정은 아니에요

사용할때 트리를 설정을 안해두고 보통 fzf 로 파일을 찾아 다니기 때문에 fzf 가 필요해요

windows

```sh
winget install fzf fd ripgrep
```

cachy os (arch linux)

```sh
sudo pacman -S fzf ripgrep fd
```
